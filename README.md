# Variant 设计方法 Skill

> 不问需求，直接展示。像 Variant 一样工作。

## 理念

"Most tools make you explain what you want. Variant just shows you."

不要让用户描述需求，直接展示多个方案，通过选择学习审美。

## 特点

- ⚡ 快速生成 3-5 个完全不同的方案
- 🎨 多样性优先（风格、重点、调性都不同）
- 🔄 快速迭代（说"再来"就继续生成）
- 🧠 学习偏好（基于选择调整方向）

## 使用

直接说你想要什么，我会立刻给你多个方案：

```
用户：BotEarn 落地页
AI：[直接给 4 个完全不同的方案]
```

## 适用场景

- 落地页设计
- 文案方向
- 视觉风格
- 产品定位
- 营销策略
- 内容创作

## 原则

✅ 立刻生成，不问细节
✅ 每个方案差异明显
✅ 简洁呈现，易于比较
✅ 基于选择快速迭代

❌ 不问太多问题
❌ 不生成相似方案
❌ 不过度解释
❌ 不强行推荐
